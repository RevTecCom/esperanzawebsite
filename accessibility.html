Accessible Fonts

We use two fonts that are aesthetically pleasing, while being easy to read for people with low vision or dyslexia.

For normal text, we use Atiknson Hyperlegible, a free font developed by the Braille Institute for people with low vision. It has versions optimized for both print and the web. Web developers can easily access it by using Google Fonts.

For code and command excerpts, we use Intel One Mono, a free monospaced font developed to be readable even by those coders who are legally blind.

Font Sizes

To increase readability, we use a 14pt size for normal text, a 15-16pt size for titles, and a 12pt size for small text.
